Screentool - Screen Recording Manager

Screentool captures screen recordings using ffmpeg. Recordings are stored in session directories under $ST_DIR.

Recording Sessions

Each session is identified by a timestamp-based ID and stored as a directory containing the video file and optional audio.

$ST_DIR/ 1234567890/ screen.mkv audio.wav

Info

Click any recording row to expand and see ffprobe details: codec, resolution, bitrate, and duration.

Status

The header shows recording status. A red indicator appears when ffmpeg is actively recording.
